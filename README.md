**FinSight** is a high-fidelity financial management dashboard designed to provide real-time insights into spending habits, income streams, and savings goals. Built with a focus on **Calm Design** principles and a custom **Mint-Visual Language**, it transforms raw transaction data into actionable financial intelligence.

---

##  Key Features

* **Interactive Data Visualization:** Real-time tracking of Balance, Income, and Expenses using dynamic Bento-style charts.
* **Smart Trend Analysis:** Logic-based color coding (Emerald/Red) to instantly identify budget health and spending spikes.
* **Gamified Savings:** Progress-tracking rings and milestone streaks to encourage consistent financial habits.
* **Responsive Architecture:** Fully optimized for all device sizes using a fluid Tailwind-grid layout.
* **Custom Branding:** Personalized UI featuring a Rupee-centric identity and a sophisticated Mint-tinted interface.

##  Technical Stack

-   **Frontend:** React.js with Vite (Lightning-fast HMR)
-   **Styling:** Tailwind CSS (Utility-first CSS) & Shadcn UI
-   **State Management:** React Context API (FinanceProvider)
-   **Icons:** Lucide-React
-   **Deployment:** GitHub Pages / Vercel

##  Project Structure

-   `src/components/`: Modular UI elements (SummaryCards, SpendingBreakdown, Header).
-   `src/context/`: Centralized financial state and theme logic.
-   `src/data/`: Mock financial datasets customized for regional contexts.
-   `src/pages/`: Main dashboard views and 404 handling.
-   `public/`: Custom branding assets including the FinSight Rupee icon.

##  Setup & Installation

To run the project locally, ensure you have [Node.js](https://nodejs.org/) installed:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/piyushjaiswal2-sys/warm-wealth-watch.git](https://github.com/piyushjaiswal2-sys/warm-wealth-watch.git)
    cd warm-wealth-watch
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```
